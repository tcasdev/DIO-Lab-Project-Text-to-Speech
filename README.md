# DIO-Lab-Project-Text-to-Speech
Este repositório faz parte do projeto Conversando por Voz Com o ChatGPT Utilizando Whisper (OpenAI) e Python da DIO, em que o aluno poderia selecionar uma parte do projeto para fazer o seu próprio.

A parte escolhida foi utilizar a biblioteca gTTS para converter texto em fala, gerando um arquivo de áudio em português. A frase escolhida foi extraída do livro memórias Póstumas de Brás Cubas de Machado de Assis, em que esta clássica frase faz uma reflexão sobre a memória e a passagem do tempo, proporcionando uma experiência auditiva envolvente e com significado profundo.
